# Novel Writing Skill

小说创作完整工作流：规划 → 初稿 → 审阅 → 改进 → 完成

## 核心小说要素清单

| 要素 | 要求 |
|------|------|
| 人物 | 主角有明确动机与内在矛盾，配角有独特个性 |
| 情节 | 有冲突、有悬念、有节奏，起承转合清晰 |
| 主题 | 有核心立意，不是单纯叙事 |
| 对话 | 性格化，符合人物背景 |
| 情感 | 能引发读者共鸣，有情感起伏 |
| 视角 | 统一且适合故事 |

## 写作技巧（必须运用）

### 1. 闪回技巧
- 用回忆片段揭示人物背景和情感，200-300字足以
- 每人最多3-4次闪回，时机：人物弧线转变/真相揭露/情感高潮
- 常见触发：场景中的某个物品、声音、气味
- 闪回触发物必须提前埋线

### 2. 内心描写
- 主角必须有内心挣扎，不能只是行动机器
- 用短句表现犹豫、痛苦
- 用内心独白深化主题

### 3. 对话精简
- 对话要有信息量或有张力，不能废话
- 潜台词要大于表面意思
- 打断、沉默、停顿都是对话的一部分
- 紧张场景用短句+省略号，慢场景可以稍长
- 每句对白不超过20字

### 39. 人物关系设计
- 每个主要角色要有明确的关系线（亲人/敌人/恋人等）
- 关系要有变化和发展
- 反派和主角可以有共同点，形成镜像

### 4. 悬念设置
- 章末钩子：留一个未解之谜
- 诱饵信息：给读者部分真相让他们追问
- 延迟满足：不要马上揭晓，让读者焦虑

### 47. 章节结尾的5种类型
- **悬念式**：留问题，吸引继续读
- **转折式**：突然改变情况
- **情感式**：情感达到某个点
- **意象式**：用物件/场景收尾
- **静默式**：突然结束，无声响
- 每章结尾类型要变化，不能总用一种

### 48. 并发章节合并指南
- 多批次完成的章节需要合并
- 合并时检查：
  - 章节编号是否连续
  - 内容是否有重复
  - 风格是否统一
  - 伏笔是否连贯
- 合并后做一次总检查

### 51. 章节质量检查
- 每5章回头看：埋的线有回应？悬念推进？情感连贯？
- 每10章总检查：重复内容？断开的情感线？忘记的伏笔？节奏断层？
- 读某章走神→重写
- 连续5章同一节奏→调整

### 66. 章节编号与格式
- 编号必须连续：1,2,3...无重复无漏号
- 格式统一：`# 第X章：标题`，禁用`# 第八十一章`
- 并发写作最容易出错

### 67. 章节连贯性
- 检查：时间/地点/人物/情感是否衔接
- 不连贯用过渡章节补救

### 72. 100章章节润色技巧
- 润色优先级：
  1. 第一章（决定读者是否继续）
  2. 最后一章（决定读者是否满意）
  3. 每10章的第一章
- 润色重点：
  - 开头50字是否抓人
  - 结尾50字是否有余韵
  - 对话是否性格化

### 73. 100章写作常见错误
- 常见错误清单：
  1. 开头不抓人
  2. 对话过多
  3. 场景重复
  4. 人物扁平
  5. 节奏拖沓
  6. 悬念过度
  7. 结尾说教
- 每写完10章自查一次

### 52. 章节修改优先级
- 修改优先级排序：
  1. 人物弧线问题（最优先）
  2. 情节漏洞
  3. 主题表达
  4. 对话问题
  5. 场景描写
  6. 字数调整
- 每次修改不要超过3章

### 5. 情感深化
- 用具体细节代替抽象情感词
- 不要写"他很悲伤"，写"他把烟头按灭在手掌上"
- 结尾有余韵，不要把话说满
- 每20章要有一次情感升温
- 情感冲突要有层次：表面→深层→价值观碰撞

### 54. 100章情感高潮设计
- 主要情感高潮点：
  - 第20章左右：兄弟分离的痛苦
  - 第50章左右：重逢的希望
  - 第80章左右：生死的抉择
  - 第100章：无声的告别
- 情感高潮需要铺垫（至少3章）

### 6. 场景氛围描写
- 每章至少有一个感官细节（视觉/听觉/嗅觉/触觉）
- 用环境暗示情绪，不要直接说"他很紧张"
- 场景转换要有过渡，不要跳跃
- 潜入/逃亡场景要多用触感：心跳、呼吸、汗湿的手心

### 7. 伏笔与揭露技巧
- 重要信息要提前埋线（至少2-3章前）
- 身世揭露、前史揭露需要铺垫，不能太突兀
- 揭露方式：闪回、对话揭示、文件/证据触发

### 8. 配角塑造
- 每个配角至少有一个独特习惯/口头禅
- 配角要有自己的动机，不只是工具人
- 用配角视角补充主角看不到的信息

### 9. 章节间的钩子呼应
- 前面埋的线索后面要呼应
- 人物习惯在多个场景保持一致
- 同一个物品/细节在不同章节出现，形成暗线

### 10. 开篇抓人技巧
- 第一句话就要有钩子
- 可以用行动开场、氛围开场、或悬念开场
- 不要用"很久以前"、"有一天"这种俗套开头
- 开头就要建立紧张感或吸引力

### 11. 紧张场景节奏控制
- 紧张场景用短句，快节奏
- 环境描写要少，聚焦在人物反应上
- 用感官细节代替长叙述：声音、触感、温度
- 省略号表示时间停顿，制造紧张感
- 不要解释太多，让读者自己感受

### 12. 反派复杂性塑造
- 反派不是纯粹的恶，每个反派都有自己的理由
- 反派有自己的恐惧、弱点、和牺牲
- 反派和主角可以有共同点，形成镜像关系
- 反派可以说出主角不想听的话，有几分道理
- 让读者在某个瞬间同情反派，但不能让他洗白

### 50. 反派塑造的三个阶段
- **第一阶段（1-30章）**：神秘阴影
  - 反派不露面，只通过行动影响
  - 让读者感受到反派的威胁
- **第二阶段（31-70章）**：揭露复杂
  - 反派登场，展现动机
  - 让读者理解但不认同
- **第三阶段（71-100章）**：最终对决
  - 反派与主角正面冲突
  - 结局要体现反派的最终立场

### 13. 结局余韵处理
- 结局不要总结，要留白
- 用一个具体的物件/动作/场景收尾，不要用总结句
- 结局要回应开篇，形成闭环
- 最后一个句子要有重量，要让读者停下来思考
- 不要写"从此他们幸福地生活"这种废话

### 14. 5000字章节强制结构

**每章3000字分配**：

**检查**：
- [ ] 总计≥3000字？
- [ ] 感官3个+？
- [ ] 对话3段+？
- [ ] 闪回2个+？
- [ ] 钩子结尾？
- **场景A**（300-400字）：开篇，必须包含：时间地点人物、3个以上感官细节、情绪基调
- **闪回A**（300-400字）：触发物→回忆场景→200字情感细节
- **场景B**（300-400字）：核心事件，对话5段以上，每段3-10句
- **闪回B**（300-400字）：深化情感，100字内心独白
- **场景C**（300-400字）：第二事件，对话3段+
- **闪回C/内心**（300-400字）：100字内心独白
- **场景D**（400-500字）：高潮，必须有冲突升级
- **结尾钩子**（200-300字）：悬念/意象收尾
